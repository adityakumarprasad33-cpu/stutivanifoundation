# Stuti-Vani Foundation
# Product Requirements Document (PRD)

Version: 1.0

---

# Introduction

The purpose of this document is to define every functional and non-functional requirement of the Stuti-Vani Foundation platform.

This document acts as the single source of truth for developers, designers, AI coding assistants, and future contributors.

Every feature implemented in the project must originate from this document.

---

# Product Goal

Build a premium, modern, secure, scalable NGO platform that enables:

• Trust
• Transparency
• Donations
• Volunteer Participation
• CSR Partnerships
• Awareness
• Community Engagement

---

# User Types

## Visitor

Can

- Browse website
- View projects
- Read blogs
- Donate
- Contact NGO
- Register as volunteer
- View reports

Cannot

- Edit content
- Access dashboard

---

## Volunteer

Can

- Login
- Update own profile
- View assigned activities
- Receive announcements

Cannot

- Modify public content

---

## Admin

Can

- Manage projects
- Manage blogs
- Manage gallery
- Manage events
- Manage volunteers
- Manage contact messages

---

## Super Admin

Full access to every feature.

Can

- Create Admins
- Delete Admins
- Change website settings
- View analytics
- Manage permissions
