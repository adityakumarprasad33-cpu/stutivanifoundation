# Stuti-Vani Foundation
# Design System

Version: 1.0

---

# Purpose

The Design System defines every visual rule used throughout the Stuti-Vani Foundation platform.

Its purpose is to ensure visual consistency, accessibility, scalability, and maintainability.

Every UI component, page, and interaction must follow this document.
